Getting started with Blender


If you are interested in anything 3D, you might know about Blender. It is an open-source piece of code that houses all the pipelines related to the artisitic endeavour of creating 3D animations. It's paid counterpart includes Autodesk Maya (which has a free Student Version). 

Here, I intend to link various sources that I used to learn basics of Blender. At the beginning, it can be tricky and very annoying. For me it was very non-intuitive but the deeper you go, you start to understand the reasoning behind its tedious controls.

Here is the link to download Blender. I often use the beta versions as they rarely have any software-breaking bugs. Plus, it keeps you updated on the ever-changing UI of Blender.

A good place to begin would be these tutorials by BlenderGuru.

If you want a more fast-paced tutorial to create abstracts, I highly recommend Ducky3D.
